在Windows11上安装Ubuntu子系统

发表于2024-03-17|更新于2024-03-17|教程
|字数总计:320|阅读时长:1分钟|阅读量:

前言

最近毕设到最关键的一步了，就是模型训练，根据RWKV Runner的训练提示，需要安装WSL子系统，那就装吧，反正也没坏处。

开始

开启WSL

控制面板 -> 程序 -> 程序和功能 -> 启用或关闭Windows功能，点击确定后重启电脑

打开终端，输入wsl --set-default-version 1，切换WSL版本

安装Ubuntu

打开微软应用商店，搜索ubuntu，我这里下载Ubuntu 20.04.6 LTS版本，点击安装后稍作等待

下载完成后点击打开，则进行自动安装，安装完毕后需要输入用户名、密码以及重复密码，输入完成后即可进入系统：

修改root密码

在ubuntu系统中输入sudo passwd root以修改root密码。

如果你想在使用sudo时不输入用户密码，可以使用以下操作取消密码输入：

  1. 使用sudo visudo，编辑权限文件。
  2. 找到%sudo ALL=(ALL:ALL) ALL，更改为%sudo ALL=(ALL:ALL) NOPASSWD: ALL。
  3. 先使用ctrl+s组合键进行保存，然后使用ctrl+x组合键关闭文件。

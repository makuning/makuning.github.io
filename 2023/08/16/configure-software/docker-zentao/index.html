安装禅道

访问[服务器ip]:[映射容器80端口的主机端口]，记得放行防火墙。

点击开始安装，同意协议后点击下一步

编辑配置，这里只用修改root密码

预览配置后点击下一步，我这里选择使用全生命周期管理模式

设置公司、管理员账号以及密码，不熟悉可以导入demo数据，点击保存

保存后会提示我们注册一个社区账号，可以不用注册，直接点击登录按钮跳转到登录页面，并输入上面设置的管理员账号以及密码

点击登录进入系统，成功进入后台，即表示安装成功
